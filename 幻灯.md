# 页
```latex
\documentclass{beamer}
\begin{document}
\begin{frame}{title}
content
...
\end{frame}
\end{document}
```
# 结构
标题页
```
\begin{frame}
\title{}
\author{}
\date{}
\maketitle
\end{frame}
```
目录页
```
\frame{\tableofcontents}
```
显示当前进程,高亮当前节
```latex
\frame{\tableofcontents[currentsection]}
```
# 特效
使用\pause命令
```latex
\begin{frame}{受持此经，功德无量}
\begin{itemize}
	\item 初日分以恒河沙等身布施
	\pause
	\item 中日分复以恒河沙等身布施
	\pause
	\item 后日分亦以恒河沙等身布施
	\pause
\item 如是无量百千万亿劫以身布施
\end{itemize}
\end{frame}
```
